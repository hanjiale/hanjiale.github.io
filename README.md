# Jiale Han Personal Website

This repository contains the source for [hanjiale.github.io](https://hanjiale.github.io), a Jekyll-based academic personal website.

## Local Preview

Install the Ruby dependencies, then serve the site locally:

```bash
bundle install
bundle exec jekyll serve
```

The site will be available at `http://localhost:4000`.

## Structure

- `_pages/`: main site pages
- `_data/navigation.yml`: top navigation
- `_config.yml`: site metadata and author profile
- `assets/css/main.scss`: theme imports and local visual refinements
- `images/`: profile and site images
- `files/`: downloadable files
